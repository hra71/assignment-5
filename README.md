1️⃣ Difference between var, let, and const

var, let, and const are used to declare variables in JavaScript but they behave differently. var has function scope and can be reassigned and redeclared, while let has block scope and can be reassigned but not redeclared. const also has block scope but its value cannot be reassigned after declaration.

2️⃣ What is the Spread Operator (...)?

The spread operator (...) is used to expand or unpack elements of an array or properties of an object. It is commonly used to copy arrays, merge arrays, or clone and extend objects. This makes code shorter and helps avoid modifying the original data.

3️⃣ Difference between map(), filter(), and forEach()

These are array methods used to process elements of an array. map() transforms each element and returns a new array, while filter() selects elements based on a condition and also returns a new array. forEach() simply executes a function for each element but does not return a new array.

4️⃣ What is an Arrow Function?

An arrow function is a shorter syntax for writing functions in JavaScript using =>. It makes the code more concise and readable, especially for small functions. Arrow functions are commonly used in callbacks and modern JavaScript programming.

5️⃣ What are Template Literals?

Template literals are a way to create strings in JavaScript using backticks (`). They allow variables and expressions to be embedded directly inside a string using ${}. They also make it easier to write multi-line strings and dynamic text.
